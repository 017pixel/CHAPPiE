8K Context-Fenster Optimierung

CHAPPiEs Steering-Server (Qwen3.5-4B) wurde auf 8K max_position_embeddings begrenzt, um CUDA-OOM auf der Tesla T4 zu verhindern. Dieser Plan optimiert die gesamte Prompt-Pipeline, damit der Agent auch mit 10–15 Long-Term-Memories, Short-Term-Memory und langen Chat-Sessions garantiert unter 8K Tokens bleibt.

def build_prompt(self, system, memories, user_input, history=None, max_history=20):
    messages = [Message(role="system", content=full_system)]
    if history:
        # NUR die letzten max_history Messages
        recent = history[-max_history:] if len(history) > max_history else history
        for msg in recent:
            messages.append(Message(role=msg["role"], content=msg["content"]))
    messages.append(Message(role="user", content=user_input))
    return messages

def _consolidate_ltm_for_prompt(self, memories: list, user_input: str) -> str:
    """Konsolidiert 6+ RAG-Memories via Cerebras zu einem kompakten Paragrafen."""
    if len(memories) <= settings.ltm_consolidation_threshold:
        return self.memory_engine.format_memories_for_prompt(memories)  # original

    # Baue Prompt mit allen Memory-Inhalten
    entries = "\n---\n".join(f"[{m.role}] {m.content}" for m in memories)
    prompt = (
        "Fasse diese Erinnerungen fuer CHAPPiE zu EINEM kompakten Paragrafen zusammen.\n"
        "Extrahiere: Fakten ueber den Nutzer, wichtige Entscheidungen, emotionale Muster,\n"
        "aktuelle Kontexte. Keine Aufzaehlung – ein fliessender Text, max 600 Zeichen.\n\n"
        f"{entries}"
    )
    brain = CerebrasBrain(model=settings.cerebras_model)  # llama-3.1-8b
    config = GenerationConfig(max_tokens=300, temperature=0.1, stream=False)
    result = brain.generate([Message(role="user", content=prompt)], config=config)
    return "=== KONSOLIDIERTE ERINNERUNGEN ===\n\n" + result.strip()
API-Call nur wenn >5 Memories. ~$0.0002 pro Call. Reduziert ~1250 Tokens auf ~250.

Methode in backend_wrapper.py (vor brain.generate() aufrufen):

def _enforce_context_budget(self, messages: list, system_prompt: str) -> tuple[list, dict]:
    """Prueft Token-Budget und trimmt bei Ueberschreitung. Return (messages, budget_info)."""
    estimated = self._estimate_total_tokens(messages)
    budget_info = {"estimated_tokens": estimated, "was_trimmed": False, "trim_details": []}

    if estimated <= settings.context_token_warning_threshold:
        return messages, budget_info

    if estimated > settings.context_token_limit:
        # Trim-Prioritaet: Context-Files > STM > Alte History-Messages
        trimmed, details = self._trim_messages_by_priority(messages, settings.context_token_limit)
        budget_info["was_trimmed"] = True
        budget_info["trim_details"] = details
        return trimmed, budget_info

    budget_info["near_limit"] = True
    return messages, budget_info

def _estimate_total_tokens(self, messages: list) -> int:
    # Grobe Schaetzung: 1 Token ~ 4 chars (wie CerebrasRateLimiter)
    total_chars = sum(len(str(m.get("content", ""))) for m in messages)
    return total_chars // 4

Trimming-Priorität (niedrigste zuerst kürzen):

6. Context-Files (soul/user/prefs)→ kürzen auf 500 Zeichen je
5. STM-Einträge→ auf 10 Einträge reduzieren
4. Ältere History-Messages→ älteste zuerst entfernen
3. RAG-Memories (konsolidiert)→ behalten
2. Aktuelle User-Input→ immer behalten
1. System-Prompt (Core)→ immer behalten
