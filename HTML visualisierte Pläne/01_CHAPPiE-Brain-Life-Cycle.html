CHAPPiE Plan #01 v13.x

Brain Cycle & Life Simulation
bei einer Nutzereingabe

Vom Input über 7 Gehirn-Agenten, Global Workspace, Steering und LLM-Call bis zur Life-Finalisierung. Jede Phase trägt zu CHAPPiEs bewusster Antwort bei.

Gesamtzyklus

13 Phasen

Uhrzeit, Baseline-Decay, Aktivität, Homeostasis, Snapshot

Brain Sensory Cortex Phase 2

Input-Klassifikation: Typ, Dringlichkeit, Memory-Bedarf

Brain Amygdala Phase 3

7 Emotionen analysieren, Intensity, memory_boost, steering_hints

Brain Hippocampus Phase 4

Encoding, Query-Extraktion, Context-Relevanz, Short-Term-Entries

Memory Memory Engine Phase 5

Episodische Suche mit emotional_boost + context_relevance

GW Global Workspace Phase 6

7 Signale mit Salience bündeln & priorisieren → Broadcast

Brain Prefrontal Cortex Phase 7

Strategie, Tone, Guidance, Planning-Mode, Action-Plan

Steer Steering Manager Phase 8

VAD → Alpha → Composite Modes → Layer Editing / Style-Injection

LLM LLM-Call Phase 9

1 Call mit Prompt + Steering-Payload → generierte Antwort

Out Antwort + Debug Phase 10

Token-Level Streaming, Debug-Daten, Causal Trace

Life finalize_turn() Phase 11

Goal-Progress, Relationship, Habits, Attachment, Self-Model, Timeline

BG Background Processing Phase 12

Basal Ganglia (Reward), Neocortex, Memory Agent + Sleep Check

Brain Pipeline — Phasen im Detail

01

Sensory Cortex

Klassifikation

Bestimmt Input-Typ (conversation, command, question), Dringlichkeit (low/medium/high/critical) und Memory-Bedarf.

▶ Code-Auszug (brain/agents/sensory_cortex.py) 
input_type = classify(user_input)
urgency = detect_urgency(user_input)
memory_needed = needs_memory_retrieval(input_type, urgency)
02

Amygdala

Emotion

Analysiert 7 Emotionen (happiness, trust, energy, curiosity, frustration, motivation, sadness) mit Intensity und memory_boost.

▶ Code-Auszug (brain/agents/amygdala.py) 
emotions = analyze_emotions(user_input, sensory)
intensity = calc_intensity(emotions)
memory_boost = emotional_weights(emotions)
steering_hints = build_steering_hints(emotions)
03

Hippocampus

Memory

Extrahiert search_query, bewertet context_relevance für soul/user/preferences, erzeugt short_term_entries.

▶ Code-Auszug (brain/agents/hippocampus.py) 
query = extract_search_query(user_input, emotions)
relevance = calc_context_relevance(query)
entries = build_short_term(user_input, query)
04

Memory Engine

Suche

Episodische Suche mit top_k + min_relevance. Amygdala liefert emotional_boost, Hippocampus context_relevance.

▶ Code-Auszug (memory/memory_engine.py) 
memories = search_memory(
    query=hippocampus.search_query,
    top_k=config.memory_top_k,
    min_relevance=config.memory_min_relevance
)
05

Global Workspace

Salience

Bündelt 7 Signale aus Brain + Life, berechnet Salience, erzeugt Broadcast für Prefrontal Cortex.

Homeostasis Need-Druck
salience = min(1.0, pressure/100)
Emotion Intensity
salience = min(1.0, intensity)
Sensory Urgency
map: low 0.25 – critical 0.95
Memory Count
salience = min(0.92, 0.38 + n*0.08)
Goal Priority
min(0.96, priority * (1.15 - progress))
World Model Confidence
min(0.9, confidence + risk*0.05)
Planning Confidence
min(0.88, conf + bottlenecks*0.04)
Forecast Risk
0.66 (low) / 0.79 (mid/high)
06

Prefrontal Cortex

Strategie

Bestimmt response_guidance, tone, planning_mode und baut den finalen action_plan.

▶ Code-Auszug (brain/agents/prefrontal_cortex.py) 
strategy = build_response_strategy(global_workspace)
action_plan = action_response.build_action_plan(
    prefrontal, life_context, global_workspace
)
prompt_suffix = action_response.build_prompt_suffix(
    prefrontal, life_context, global_workspace
)
07

Steering Manager

VAD+Alpha

Mapping der 7 Emotionen auf VAD (Valence/Arousal/Dominance), Alpha-Berechnung, Composite Modes und Layer-Profile.

Lokal Forward Pre-Hook: hidden += α · vec
Cloud Style-Instruction im Systemprompt
08

LLM-Call

Generierung

Ein einziger Call mit komplettem Kontext: Systemprompt + Memory + Life-State + Steering-Payload.

▶ Provider (brain/vllm_brain.py, brain/cerebras_brain.py, ...) 
# Lokal (vLLM mit Layer Editing)
response = vllm.generate(prompt, steering_vectors=steering_payload)

# Cloud (Cerebras)
response = cloud.generate(prompt, system_hint=steering_style)

Life — prepare_turn vs. finalize_turn

VOR dem Brain

prepare_turn

  • ▶ Clock sync → Europe/Berlin (Tag + Minuten)
  • ▶ Baseline Decay: Energy -3, Curiosity -2, Rest -3, …
  • ▶ Choose Activity: aus Input + dominant Need
  • ▶ Activity Recovery: Needs verbessern sich
  • ▶ Refresh Cognitive State: Habits, Goals, World Model, Attachment, Development, Planning, Forecast, Social Arc
  • ▶ Build Snapshot: kompletter Life-State als Dict
NACH dem Brain

finalize_turn

  • ▶ Goal Progress: Keywords im Input erkennen → Progress +0.03/0.04
  • ▶ Relationship Update: closeness + trust basierend auf Ton
  • ▶ Habit Reinforcement: Aktivität + Modus verstärken Habits
  • ▶ Attachment Model: Bindungstyp neu evaluieren
  • ▶ Self-Model Update: aus Prefrontal + Global Workspace
  • ▶ Event loggen + Timeline Checkpoint + Snapshot speichern
